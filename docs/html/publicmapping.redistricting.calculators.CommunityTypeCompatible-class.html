Class CommunityTypeCompatible

source code

This calculator determines if all districts in a plan are type- compatible. A type-compatible community map contains the same community types in across all districts.

This calculator requires a "community_map_id" argument, which is the primary key of the community map (Plan) that is being compared to the Plan and a "version" argument indicating the version number of the community map.

This calculator will only operate on a plan.

Evaluate a Plan to determine if it is type-compatible. A Plan will be type-compatible if the type provided is in all Districts of the plan.

Parameters:
  • plan - The Plan to compare against the community map.
  • community_map_id - The ID of the community map.
  • plan_version - Optional, The version of the plan. Defaults to the latest version of the plan.
  • community_version - Optional. The version of the community map. Defaults to the latest version of the community map.
  • type - The community type to check for compatibility.
