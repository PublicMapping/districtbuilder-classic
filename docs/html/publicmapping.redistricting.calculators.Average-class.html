Class Average

source code

Calculate the average of a series of values.

This calculator will add up all arguments passed into it, and return the mathematical mean of their values.

For a district, this calculator will average a set of scores for each district.

For a plan, this calculator will average a set of district scores across the entire plan.

Each argument should be assigned the argument name "valueN", where N is a positive integer. The summation will add all arguments, starting at position 1, until an argument is not found.
