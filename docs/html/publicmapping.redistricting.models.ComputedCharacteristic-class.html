Class ComputedCharacteristic

source code

ComputedCharacteristics are cached, aggregate values of Characteristics for Districts.

ComputedCharacteristics represent the sum of the Characteristic values for all Geounits in a District. There will be one ComputedCharacteristic per District per Subject.
